The active implementation uses byte-address row matching and selected-cursor full-memory trace evidence: row-level read/write replay soundness proves projected `TraceReplaySound`, concrete memory-bus replay proves Sail/replay cursor agreement by prefix induction, and load arms prove `LoadMemoryBurden` from accepted row-trace construction. The active implementation replaces raw per-load `MemoryTraceAgreement` promises with accepted AIR/Main/Mem row-trace construction, FullEnsemble-shaped selected Mem read-replay row coverage, and Sail/replay cursor agreement; selected read tags are now derived from each load envelope's Main `bMem` match. The remaining blocker is proving those inputs from accepted full execution data, including shared trace/table embedding, selected envelope Mem-row occurrence, selected prefix cursor coverage, prefix-read soundness, and duplicate-free chronological rows.
